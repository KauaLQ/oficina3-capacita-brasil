# Oficina 3 | Capacita Brasil 
**Olá, WebDev's!** Que tal apertarem os cintos e embarcarem comigo pelos **lugares que eu gostaria de conhecer**? Este é o tema da oficina número três do curso de Desenvolvimento FullStack | Capacita Brasil.
